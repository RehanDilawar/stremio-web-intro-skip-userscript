# Stremio Web Skip Intro (Official Web Port)
A user script for stremio web to have skip intro plugin that was originally in Stremio Enhanced
# NOTE
This is not an addon but userscript and can only be used in [stremio web](https://web.stremio.com/) and [stremio web](https://web.strem.io/)

# INSTALLATION


## Tampermonkey

First of all you need to have [tampermonkey](https://www.tampermonkey.net/) or Violentmonkey installed on your browser to be able to install userscripts

## Greasy Fork

You can find the scrip [here](https://greasyfork.org/en/scripts/560968-stremio-web-skip-intro-official-web-port)
